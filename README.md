# Introduction
技术栈：react + redux + react-router + express + Nginx

练习点：
- redux 连接
- react-router 路由跳转
- scss 样式书写
- 容器组件与展示组件的设计
- express 脚手架项目结构设计
- 用户信息持久化（cookie + redis）
- 常见安全问题处理（xss sql 注入 cookie 跨域）
- Promise 封装数据库操作
- PM2 进程守护
